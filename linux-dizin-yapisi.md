# Linux Dizin Yapısı Notları

| Dizin | Açıklama |
|---|---|
| /etc | Sistem yapılandırma dosyaları |
| /home | Kullanıcıların kişisel dizinleri |
| /var | Değişken veriler (loglar, geçici veri) |
| /usr | Kullanıcı programları ve kütüphaneleri |
| /bin | Temel çalıştırılabilir komutlar |
| /tmp | Geçici dosyalar |
| /root | Root kullanıcısının ev dizini |
| /dev | Donanım aygıtları |
| /proc | Çalışan süreçler hakkında bilgi |
